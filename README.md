# League of Legends Performance Analytics

A static dashboard for analyzing League of Legends match performance data.

## Live Site

This repository is configured for GitHub Pages. The dashboard is available at:

`https://[username].github.io/[repo-name]/`

## Repository Contents

This repository contains only the files necessary for the static website:

- `index.html` - Main dashboard page
- `badges.js` - Achievement badge system
- `data-loader.js` - Analytics engine and data loading
- `players.json` - Player configuration
- `data/` - Match data in CSV format (one folder per player)
- `.nojekyll` - Disables Jekyll processing

## Development Files

Development files (Python scripts, documentation, etc.) are excluded via `.gitignore`.
